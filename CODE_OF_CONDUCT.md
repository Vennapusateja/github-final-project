# Code of Conduct

## Our Pledge
We pledge to make participation respectful and harassment-free.

## Our Standards
Examples of acceptable behavior include respectful language and collaboration.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing standards.

## Scope
This Code of Conduct applies within project spaces and public interactions.

## Enforcement
Instances of abuse may be reported to the project team.

## Enforcement Guidelines
Consequences range from warnings to permanent bans.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
